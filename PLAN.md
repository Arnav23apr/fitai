# Resume Workout Button & Dynamic Island Workout Icon

## Changes

### 1. Resume Workout Button (Plan Tab)

- When a workout session is active, the "Start Workout" button in the Today's Goal card transforms into a **"Resume Workout"** button
- The button turns **green** with a play/arrow icon to indicate an active session
- The Today's Goal card gets a subtle **green gradient border and background tint** to visually signal the workout is in progress
- The progress ring on the right updates live to show how many exercises are completed
- A small **elapsed timer badge** appears near the button showing how long the workout has been going

### 2. Dynamic Island — Workout-Specific Icon

- The compact leading icon (small pill) and minimal view will show the **current workout's icon** (e.g. the running figure for Legs, dumbbell for Arms, etc.) instead of the generic dumbbell
- The expanded view already uses the workout icon — no change needed there
- The lock screen Live Activity banner will also use the workout-specific icon consistently
